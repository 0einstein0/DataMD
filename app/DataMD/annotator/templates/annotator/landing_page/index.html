About Us

Medical institutions generate large volumes of biomedical image data. This can be used to train supervised models for valuable AI-assisted diagnosis in healthcare. However, annotating biomedical images is a laborious process that also needs input from professionals. With DataMD users can smoothly complete annotations with an optimized user interface, and an active learning architecture built upon a pre-trained machine learning model will use uncertainty and diversity sampling strategies alongside annotation predictions to substantially reduce the labor.
